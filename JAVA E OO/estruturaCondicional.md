> ### **Estrutura Condicional**

```java
//SIMPLES:

    if(<condição>){
        <comando1>
        <comando2>
    }
```
<br>

```java
//COMPOSTA:

    if(<condição>){
        <comando1>
        <comando2>
    } else {
        <comando3>
        <comando4>
    }

```
<br>

```java
//Encadeamento:

    if(<condição>){
        <comando1>
        <comando2>
    } else if(){ 
        <comando3>
        <comando4>
    } else {
        <comando5>
        <comando6> 
    }
```
