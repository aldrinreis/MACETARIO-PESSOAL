# **Fundamentos Java**

> ### **Fundamentos de Java**

- **Comentários**

Comentários são definidos por dupla / ```//```

- **Saída no terminal**

O print é definido por: ```System.out.println("Olá mundo");```

```java
public class Main {

	public static void main(String[] args) {
		// Comentário
		System.out.println("Olá mundo");
	}

}
```
